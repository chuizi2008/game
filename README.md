V2.4[2015-8-13]
客服端模块整理规范了下.消息协议从JSON转到BYTE数组，JSON虽然方便省事，但是性能有点慢.
TcpManager为客服端SOCKET管理模块
MsgPacket   存储使用的消息包
TcpHandle   对外使用的管理器
WriteMsg    2进制消息写包
ReadMsg     2进制消息读包
下一步开始对接LUA部分了
